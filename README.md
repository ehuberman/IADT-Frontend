# IADT Frontend (POC)

Angular-based frontend prototype for generating soccer tactics, including lineups, player roles, and tactical approaches using AI.

## Tech Stack

- Angular 20  
- TypeScript  
- Tailwind CSS  

## Features

- Dynamic lineup visualization for soccer formations  
- AI-generated tactical recommendations based on selected players  
- Interactive UI for exploring different tactical approaches  

## Getting Started

```bash
npm install
ng serve
```

Navigate to `http://localhost:4200/` to view the app.

## Notes

This project is a proof-of-concept built to explore:
- Integrating AI-driven insights into a frontend application  
- Angular component architecture and state management  
- UI patterns for visualizing sports tactics  

## Future Improvements
